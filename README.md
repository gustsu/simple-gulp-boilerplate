# simple-gulp-boilerplate
GULP4 + SCSS + ES6 


## File Structure
    .
    ├── src                     # Source Files
    │   ├── js                  # All JS Files
    │   ├── scss                # All SCSS Files 
    │ 
    ├── dist                    # Minified and Transpiled Files
    │   ├── js                  # Minified JS File
    │   ├── css                 # Minified CSS File 
    └── ...

## Available Commands

| Command                | Description                  |
| ---------------------- | ---------------------------- |
| `npm install`          | Install project dependencies |
| `gulp` or `gulp build` | Builds project               |
| `gulp watch`           | Watches Files For Changes    |
